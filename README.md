# playwright-test

End-to-end testing project using Playwright

## Project Structure

```
├── tests/

│   └── tests/        # Test specifications
├── .env             # Environment variables

```

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
- Update variables as needed
